# Auto_Fill_Up_WebForm
Python Selenium 自動填寫體溫程式 機器人 爬蟲

2021年台灣新冠疫情爆發的當下，幾乎每家公司都會要求人員每天(假日也要...)量測體溫，並回填寫至網頁上，以統計人員有無體溫異常，這麼Boring and Routine的事，寫個自動填寫體溫機器人爬蟲程式來幫我自動填寫，對於懶人的我來說，實在再也方便不過了...

我的開發程式環境是:

OS:win10 x64

瀏覽器:Chrome 

IDE: Visual Studio

Python 3.6 

Selenium 3.8.0
